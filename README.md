# Douban Movie Top 250 Scraper

## Introduction
A web scraper that collects the top 250 movies from Douban.

## Features
- Scrape 250 movies from Douban top 250 list
- Extract: movie name, rating, year, country, genre
- Save data to CSV file

## How to Use
1. Install dependencies: pip install -r requirements.txt
2. Run: python main.py
3. Check douban_top250.csv for results

## Files
- main.py: scraper code
- douban_top250.csv: movie data
- requirements.txt: dependencies
